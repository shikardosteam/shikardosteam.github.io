## МАСТЕР-ПРОМПТ (скопировать в новый чат Opus 4.6):

```
ЗАДАЧА: Объединение 11 Excel-файлов в единую CRM базу + парсинг актуальных данных Instagram

КОНТЕКСТ:
Прикреплены 11 Excel-файлов. Из них:

1. CRM_Shikardos_Team_2026-05-02.xlsx — основная CRM, собрана через браузерную автоматизацию (Chrome) из моих подписок: Instagram (~400 аккаунтов), YouTube (~95 каналов), Telegram (~59 каналов). Внутри 8 листов, главный — "Аутрич" (519 строк, 22 колонки). Данные актуальные (май 2026).

2. Остальные 10 файлов — базы блогеров из сервиса trendHERO (trendhero.io), собраны 1-2 года назад со скриншотов. Структура: #, Имя, Логин, Ссылка Instagram, Подписчики, ER. Файлы по нишам:
   - SMM_база_блогеров.xlsx — 209 записей
   - Маркетологи_база_блогеров.xlsx — 332 записи
   - Маркетологи_2_база_блогеров.xlsx — 315 записей
   - Таргетологи_база_блогеров.xlsx — 215 записей
   - New1_база_блогеров.xlsx — 154 записи
   - Арбитраж_трафика_база_блогеров.xlsx — 263 записи
   - Лидеры_мнений_база_блогеров.xlsx
   - Личный_бренд_эксперт_база_блогеров.xlsx
   - Продвижение_и_продажи_база_блогеров.xlsx
   - Рилсмейкеры_база_блогеров.xlsx

ВАЖНО: Данные trendHERO устарели (1-2 года). Подписчики и ER неактуальные. Некоторые аккаунты могли сменить нишу, быть заблокированы или проданы.

ЭТАП 1: Объединение + дедупликация + чистка (Python-скрипт)

Создай Python-скрипт (openpyxl), который я запущу на своём компе. Bash в Cowork не использовать.

Что делает скрипт:
1. Читает все 11 файлов
2. Объединяет в единый список людей (ключ дедупликации — Instagram username)
3. Для дублей: объединяет данные (если у одного человека есть IG + YT + TG из CRM, и он же есть в trendHERO — берём все данные, не удаляем)
4. Каждому человеку присваивает категорию/нишу (из какого файла trendHERO он пришёл)
5. Приоритет: A (релевантная ниша + 100K+ или 3 платформы), B (релевантная ниша или 20K+), C (остальные)

Фильтрация — УДАЛИТЬ нерелевантных. Но нужно сначала просканировать bio каждого.
- Оставляем: бизнес, маркетинг, таргетинг, SMM, арбитраж, affiliate, лидогенерация, воронки, контент-мейкинг, AI/нейросети, личный бренд (если про бизнес), продажи
- Удаляем: лайфстайл без бизнес-контекста, танцы, кулинария, ремонт, пижамы, выгул собак, стоматология, косметология (не маркетинг), путешествия (просто фото)
- Если миллионник — всё равно проверить bio. Если лайфстайл-девочка в Дубае с фотками — удалять

Структура итогового Excel:
- Лист 1 "Все контакты" — единый мастер-лист со всеми колонками
- Лист 2 "Удалённые" — все кого удалил с пометкой причины
- Лист 3 "Дубли" — кто совпал между файлами
- Лист 4 "Статистика" — сводка

Все существующие колонки сохранить. Добавить: Источник (из какого файла), Ниша/Категория trendHERO.

Скрипт сохраняет в D:\ЗАГРУЗКИ\CRM_Unified_Shikardos_Team.xlsx.

ЭТАП 2: Парсинг актуальных данных Instagram (через браузер)

После Этапа 1, пройтись по каждому Instagram-аккаунту через подключение к моему Chrome. Не через Apify.

Что собираем:
- Актуальное количество подписчиков (новая колонка "Подписчики 2026")
- % роста (новая колонка "Рост %" = (новые - старые) / старые × 100)
- Актуальное bio

Если аккаунт заблокирован — пометить "Заблокирован".
Если сменил нишу (был маркетолог, стал стоматолог) — переместить в "Удалённые".
Если будут лимиты Instagram — я залогинюсь под другим аккаунтом.

Добавляем 2 колонки ТОЛЬКО к тем у кого есть ER из trendHERO (т.е. данные из trendHERO файлов).

ТЕХНИЧЕСКИЕ ПРАВИЛА:
- Excel создавать ТОЛЬКО через Python (openpyxl) на моём компе
- Bash в Cowork не использовать
- Формат: шапка #4472C4 белый жирный Arial 11, чередование #D9E2F3, ссылки кликабельные #0563C1, границы тонкие, автофильтр, freeze_panes="A2"
- Отвечать на русском

Начни с Этапа 1. Сначала прочитай все файлы, покажи что понял, потом создай скрипт.
```
