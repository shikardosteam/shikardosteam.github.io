## ЗАДАЧА: Сбор базы Instagram-блогеров из trendHERO
**СТАТУС:** Завершено (10 файлов)

**КОНТЕКСТ:**
Подписка trendHERO ($30/мес) может быть неактивна — экспорт списков невозможен. Работаем через скриншоты.

**ЧТО РАБОТАЕТ:**
1. Валерий кидает скриншоты пачками (списки блогеров из trendHERO)
2. Claude извлекает из каждого скрина: **Имя, Логин, Подписчики, ER**
3. После каждой пачки — показать пронумерованный список (для проверки)
4. Создать Python-скрипт (.py файл) с openpyxl
5. Скрипт сохраняется в рабочую папку Claude, дать Валерию полный путь
6. Валерий запускает: `python "полный_путь_к_скрипту"`
7. Excel-файл сохраняется в `D:\ЗАГРУЗКИ`

**Форматирование Excel:**

| Элемент | Стиль |
|---|---|
| Шапка | Заливка `4472C4`, белый жирный Arial 11 |
| Чётные строки | Заливка `D9E2F3` |
| Ссылки | Цвет `0563C1`, подчёркнутые, Arial 10 |
| Обычный текст | Arial 10 |
| Границы | Тонкие у всех ячеек |
| Автофильтр | На шапке |
| Закрепление | `freeze_panes="A2"` |
| Ширина колонок | A=5, B=45, C=30, D=50, E=15, F=10 |

Колонки: `#`, `Имя`, `Логин`, `Ссылка Instagram`, `Подписчики`, `ER`
Ссылка — кликабельная гиперссылка `https://www.instagram.com/{login}`
