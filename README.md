# Shikardos Team

Главный сайт + внутренний Command Center (дашборд).

## Файлы

- **`shikardosteam.html`** — главный публичный сайт (открывать через GitHub Pages)
- **`dashboard.html`** — внутренний Command Center (22 раздела). Доступ через прямой URL, для команды.

## Запуск локально

Двойной клик на `shikardosteam.html` или `dashboard.html` — откроется в браузере.

## Деплой

Автоматически через **GitHub Pages** при push в `main`.

После push сайт обновляется на `https://shikardosteam.github.io/<repo-name>/shikardosteam.html` через ~1 минуту.

## Стек

- HTML / CSS / JS (без сборщиков, статичный)
- Шрифт: Bebas Neue для заголовков
- Хостинг: GitHub Pages
- Backend данных (планируется): Telegram Bot + Apify + Google Sheets / DB
